My story

I've worked at my family's Italian restaurant for as long as I can remember. I've learned and performed every job possible, from parking cars to washing dishes, shaking cocktails, and being the manager of it all. One day my father needed to update the restaurant's online presence with a newer website and as I began the search for a website developer I thought, hey maybe this is just another job I can learn, another problem I can solve for my father...

So I began learning the fundamentals necessary to start building websites which were HTML, CSS, and JavaScript. But then after that, I found myself scraping at the surface of how APIs work so that I could build a currency converter for my Colombian family here in the States. Then my journey led me to an open-source community called ProgramEquity. There I learned all about agile methodologies while picking up and using new technologies to implement security features as well as incorparate different components for the user interface. So as my learning and my curiosity increased, I began to realize that not only could I fix this specific problem for the restaurant but I could help solve many different problems for many different people.

Now I am on the path to becoming a Full-Stack Developer. Some of the skills that I am learning and building with include working with databases, frameworks, and libraries such as React, Rails, Express, Bootstrap, Mongo, PostgreSQL, Node, Cloudinary, OAuth, AWS S3, Postman, Vue, Vuetify, Jest, GitHub, Git, OOP, SAFe Agile, and much more.
